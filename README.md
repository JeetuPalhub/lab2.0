# Jeetu Pal Portfolio

Modern, production-ready portfolio built with:

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- ShadCN-style reusable UI components
- Lucide Icons

## Quick Start

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`.

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```
